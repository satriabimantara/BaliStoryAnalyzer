BaliStoryAnalyzer

BaliStoryAnalyzer is a website-based natural language processing application developed for the purpose of analyzing narrative story texts in Balinese (Satua Bali).

Try it now!

Character Identification

BaliStoryAnalyzer has been equipped with a machine learning model that can identify all lists of characters from a given story text. The algorithm we developed can also group various aliases of each character.

Character Role Classification

The Machine Learning model in this application can also classify character roles into protagonist vs antagonist.
